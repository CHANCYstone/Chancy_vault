# **5. 毅力（7）& 决心不足（8）**

  

这两点其实是一体的：

- **一旦你下定决心，你是走得很远的人。**
    
- 所以真正关键的，是如何更快、更主动地“形成决心”。
    

  

可以给自己设一个“决心形成流程”：

1. 收集信息（限定时间，比如2周/1个月）。
    
2. 写下利弊和关键假设。
    
3. 和1–2个你信任、但不会替你做决定的人讨论。
    
4. 在自定截止日期那天，**无论如何给自己一个答案**。
    
5. 告诉两个人你的选择，让他们做“见证人”，这样你会更愿意承担结果。
    

  

久而久之，你会发现：“决心”是可以被训练的，而不是靠“某天突然被点化”。


# **2. 用“最小充分信息 + 决策截止日期”强迫自己落子**

  

你的一个模式是：

  

> 想得很多 → 看得越多越觉得不确定 → 决心迟迟下不来 → 等别人推你一把。

  

可以试着换一种方法：

- 对每一个重大问题，写下：**“我要做出决策，最少需要搞清楚哪5件事？”**
    
    比如“要不要读PhD”，那5件事可能是：
    
    1. 我真实喜欢的，是研究过程本身，还是研究带来的身份与稳定？
        
    2. 现有的科研水平与顶级PhD要求相差多远？
        
    3. 是否有至少两位潜在导师是我 genuinely 想跟的？
        
    4. 博士毕业后的几种现实出路分别是什么？
        
    5. 这条路和我想要的生活方式是否匹配？
        
    
- 给每个问题设一个**“决策截止日期”**，比如：
    
    “在2026年6月之前，我必须为‘博士 vs 选调 vs 企业’做出一个版本1.0的选择。”
    
- 在这个日期到来时，**无论信息是否完美，都必须做出一个“当前最优”的选择**，而不是继续拖。
    
    决策的质量，不是来自“无限拖”，而是来自**多次迭代**（每2–3年复盘一次）。
    

  

**3. 建立“个人决策日志”**

  

每次重大决定，记四行：

1. 我现在的选择是什么？
    
2. 我为什么这么选？我的关键假设是什么？
    
3. 我放弃了哪些选项？这些选项失败的条件是什么？
    
4. 预期结果是什么？（比如：3年内达到怎样的状态，才算这次选择是“对的”。）
    

  

一年后、三年后回来看，你会发现两点：

- 其实当时的你已经有足够的信息做出不错的判断；
    
- 真正的问题往往不是“信息不够”，而是**你怕承担“自己拍板”的责任**。
    

  

反复训练之后，你的“战略感”就不是抽象的，而是实打实建立起来。
