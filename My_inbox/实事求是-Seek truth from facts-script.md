# 📒 Note

实事求是

实事求是的目的是为了调节主观客观的关系

主观和客观不符，主观和客观的矛盾，是我们有所行动的源泉

主观上我想达到什么样的预期，必须符合客观的规律。如果不符，就会在实践之中失败。在失败过后，就要调整主观，使其符合客观规律。

也就是毛主席的这句话：“**改变这个不好的世界。这里就用的着而且必须用我们的主观指导能力**。”







# 实事求是的基本内涵

实事求是：毛主席在《[[《改造我们的学习》]]》中提到

“实事，即客观存在的一切事物”
“是，就是客观事物的内部联系”
“求，就是我们去研究”

实事，干实事的，这个实事。真实的事情，客观的事情。
是，是什么，这个客观的事情是什么？本质是什么？

实事求是的英文含义为 从事实中寻找真理

> [!NOTE]
> 好问题！在马克思主义哲学（辩证唯物论）里，“规律”的经典界定就是：**现象之间那种本质的、必然的、稳定可重复的**联系。毛泽东说“事物的**内部联系**体现了事物的**规律性**”，背后有几层意思：
> 
> 1. **“内部联系”决定“是什么、向哪里”**
>     
>     事物内部诸要素（矛盾双方、结构、功能）的相互制约与转化，决定了它的性质与发展方向。外部条件只能通过内部矛盾起作用。既然内因起决定作用，那么能长期、反复呈现的**稳定联系**就构成了“规律”。
>     
> 2. **规律 = 本质性 + 必然性 + 稳定重复性**
>     
>     偶然的、外在的、一次性的接触不叫规律；只有**内在机制**（如因果链条、结构性约束、量变—质变阈值）反复起作用，才具有可预测性和可指导性，这正是“规律”的哲学含义。
>     
> 3. **认识论上的要求：从“事实”上升到“内在联系”**
>     
>     仅罗列表面事实（“实事”）不够，必须通过调查研究与理论抽象，把散乱材料上升为**揭示内在必然性的结论**（“是”）。这也是毛强调“在马列主义一般原理指导下，从材料中引出正确结论”的原因——用科学范畴把内在联系说清楚，才谈得上规律。
>     
> 4. **方法论上的指向：具体问题具体分析**
>     
>     每一类事物都有其**特殊的内部联系**（规律的特殊性）。因此必须“从国内外、省内外……的实际出发”，在特定历史、地域、组织情境中找那条真正起作用的联系，而不是搬用书本或主观想象。
>     
> 5. **实践上的意义：能解释、更能改造**
>     
>     抓住内部联系就能**解释**为什么会那样（溯因），也能**预言**在什么条件下还会那样（演绎），从而**设计有效行动**去改变条件、改变结果。这正是“规律”之所以能成为“行动的向导”。
>     
> 
>   
> 
> 简单举三个对应的例子（把“内部联系→规律→行动”串起来）：
> 
> - 自然：物体下落并非“偶然一起掉”，而是受重力与初速度等**内在机制**制约（万有引力定律）→ 可预测轨迹与时间 → 由此设计工程方案。
>     
> - 经济：价格的短期波动背后是供需结构、成本与预期的**内在关系** → 把握结构性矛盾才能制定有效政策，而不是头痛医头。
>     
> - 教育：学生成绩≠练得多这么简单；常见的**内部联系**是“先备知识→练习质量→及时反馈→再加工”的闭环。只看到作业量（外部联系）而忽略反馈与再加工（内部联系），往往提效有限；抓住闭环规律，才能设计出高效的再教学。
>     
> 
> 
> 所以，这句话要点是：**规律并不是贴在事物外表的“经验平均数”，而是把握住它内部诸要素如何稳定地互相作用的那条“必然链”**。找到了它，才称得上“求是”；沿着它去行动，才谈得上“实事求是”。



> [!NOTE]
> 马克思的实事求是被说得太多了，如同老人的唠叨，习以为常，反而不觉其中真义。豆豆的小说之所以让人震撼，就在于把习以为常的实事求是，换成了耳目一新的见路不走，把高高在上的政治理论，拉近到生活中的一言一行。读了豆豆的小说才醍醐灌顶，原来不只国家发展要实事求是，个体的发展也要实事求是，如果社会中的每个人都见路不走了，那这个社会就离共产主义不远了。
