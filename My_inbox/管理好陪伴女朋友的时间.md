| **情境** | **做法**                          |
| ------ | ------------------------------- |
| 你白天要自习 | 每天中午吃饭前语音5分钟，定个“午安电话”           |
| 她晚上想你了 | 睡前说一句“今日总结 + 想她的一句话”            |
| 周末可放松  | 安排一个远程约会（一起追剧、看书、语音散步）          |
| 她生气了   | 不要敷衍回“我在学习”，而是回“等我20分钟，我就来专心哄你” |
✅ 核心不是陪多久，而是让她知道：**你不是“没空”，而是“留出了属于她的空间”。**

中午的时间和9:40 以后的时间都留给她
