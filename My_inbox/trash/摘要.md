### 摘要1
国外政治环境动荡不安，全球经济增长逐步放缓，国内经济下行压力持续加大，私募股权投资市场监管趋严，市场募资难，私募行业向头部机构集中，中小私募机构在当前面临巨大的生存压力，因此私募股权基金行业的发展趋势以及私募股权基金公司的投资战略发展研究变得更加迫切，具有重要的现实指导价值。本文以中小私募机构J公司为案例，研究了新经济常态下私募股权基金的外部和内部环境，提出了当前J公司“专业聚焦、有限多元”的投资战略优化方案。

本文首先明确了私募股权基金的定义、分类和组织结构，其次厘清了国内私募股权基金行业的发展历程，以及当前主要存在的问题，运用PEST模型透视了私募股权基金行业的宏观环境，从募资情况，投资情况，投后管理，退出情况四个环节分析了私募股权基金的行业环境，通过波特五力模型分析了外部环境中的机遇和挑战，最后简要介绍了企业的发展历程、主要业务现状和运作流程，总结了J公司的企业内部资源和核心竞争力，确定了J公司的内部优势和劣势。

本文基于SMC模型，首先根据J公司内部行业增长及盈利空间得出行业吸引力，根据政策风险和竞争力度得出行业风险，结合两者得出优选行业。其次依据行业优选得分作为一个纬度，结合企业内部能力适配情况，得到不同行业的策略，最后参照以上各行业未来发展前景，设计公司未来3-5年的业务组合策略，确认了“专业聚焦、有限多元”的投资战略。鉴于当前J公司投资业务资源以及相关行业的判断，确定了数字经济和新能源的行业布局，同时结合股东资源，综合考虑J公司的内部资源和核心竞争力，优化了投资阶段和投资策略，最后再次从募投管退四个环节详细阐述J公司投资战略的保障措施。

### 摘要2

平衡计分卡是一项应用非常广泛但又比较年轻的绩效管理工具，相比较于方针管 理 Hoshin-Kanri，360 度绩效考核等绩效指标法，它能够将企业的长远目标和企业的 短期目标结合起来，将长期目标转化为可实施的短期目标分阶段实现，使企业长远的 核心战略可以有效落地。除了关心企业的财务目标外，平衡计分卡还考量客户、内部 流程、学习和成长等非财务指标对企业核心战略的影响，将财务目标和非财务目标结 合起来，帮助企业识别并去除一些短视的财务指标，并从长远上保障企业的良性发展。 C 工厂是一家在华的外资企业所独资拥有的制造工厂，这家制造工厂对外是一家 独立的法人实体，对内相当于集团企业下属的制造工厂，在很多方面没有绝对的自主 权，比如该企业在对外销售、研发方面不具有绝对控制权。但是由于企业是由国企过 渡到合资企业，再从合资企业过渡到独资企业，工厂在管理水平包括战略定位和绩效 管理上相对落后。但是由于工厂以前的盈利能力非常好，这些深层次的问题被利润的 光环给遮住了。现在由于市场技术升级转换到新型的产品上时，企业原先的管理不足 充分显现出来，并且利润上出现了首次亏损。 本文通过在对平衡记分卡的学习和了解的基础上，帮助 C 工厂设立一套完整的 平衡计分卡体系。首先运用 SWOT 分析方法帮助 C 工厂确立工厂的愿景和战略方向， 再依据平衡计分卡把这些战略主题分解到财务、客户、内部流程，以及学习与成长四 个维度上，使得 C 工厂的战略地图可视化。然后通过对利益相关者的问卷调查和 SMART 原则来确定各个战略主题背后的 KPI。最后再通过 AHP 分层分析法帮助工厂 确立相对客观的 KPI 评价指标的权重，并落实平衡计分卡实施的保障措施。同时出于 C 工厂的特殊地位和经营环境，C 工厂并不完全具备教科书或经典案例中实施的完美 前提，所以本文希望通过对平衡记分卡的再理解和重新解读，给类似于 C 工厂这样 的代工企业或内部工厂实施平衡计分卡带来一些样本和探索。同时也希望 C 工厂通 过此次的平衡计分卡项目，使得工厂转型成功，为工厂将来的进一步发展奠定良好的 绩效管理的基础。
