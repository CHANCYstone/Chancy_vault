| **层级**     | **名称** | **作用**               |
| ---------- | ------ | -------------------- |
| ODS（操作数据层） | 原始日志   | 不做业务加工，只做清洗          |
| DWD（明细数据层） | 精细结构化  | 以“事件”或“事实”为粒度，一行一件事  |
| DWS（汇总数据层） | 业务宽表   | 聚合后的主题表（如用户支付总览）     |
| ADS（应用数据层） | 指标报表   | 直接对接分析、可视化层          |
| DIM（维度层）   | 维度表    | 提供枚举/属性支持，如“国家、支付方式” |
