在 Mac 连接外接显示器时，将程序坞固定到主屏幕，==可以先按下 **Option + Command + D** 键来隐藏副屏幕的程序坞，然后将鼠标移到主屏幕下方中间边缘，程序坞会再次出现，最后再按一下 **Option + Command + D** 键进行锁定==。 

具体操作步骤

1. **隐藏副屏幕上的程序坞**：在键盘上按下 **Option + Command + D**，这样副屏幕上的程序坞就会消失。
2. **将程序坞调出到主屏幕**：将鼠标光标移动到你想要显示程序坞的主屏幕下方中间边缘，等待几秒钟，程序坞就会在该屏幕上出现。
3. **锁定程序坞**：再次按下 **Option + Command + D** 组合键，即可将程序坞锁定在主屏幕上。 

其他方法

- **拖移法**：在程序坞出现时，如果它出现在副屏上，将光标移到副屏的边缘，然后向外轻扫，程序坞就会移动到主屏上，这是因为副屏程序坞不会一直显示。
