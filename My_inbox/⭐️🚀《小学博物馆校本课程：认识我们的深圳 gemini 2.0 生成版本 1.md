- [[2505《小学博物馆校本课程全局规划提示词]]



**小学博物馆校本课程：认识我们的深圳**

**课程理念：** 本课程以《古代深圳》、《近代深圳》、《深圳民俗文化》和《深圳改革开放》四本参考教材为基础，旨在通过博物馆资源与校本课程的融合，帮助小学生系统了解深圳的历史变迁、民俗文化和改革开放成就，培养他们热爱家乡、传承文化的意识，提升他们的观察力、动手能力、思考能力和合作能力。

**课程目标：**
*   **知识与技能：** 了解深圳不同历史时期的重要事件、人物、文化遗存和社会生活；掌握相关的历史、地理和文化知识；学习观察文物、收集资料、分析信息和表达观点的基本方法。
*   **过程与方法：** 经历参观、体验、制作、表演、探究等多种学习活动；培养独立思考、合作学习、解决问题的能力；提升语言表达、艺术创作和实践操作的技能。
*   **情感态度与价值观：** 增强对家乡深圳的认同感和自豪感；培养热爱祖国、尊重历史、传承文化的意识；树立积极向上、勇于探索的精神。

**课程结构：**
本课程按照小学六年学段，为每个年级设计20课时的内容，每学期10课时。课程内容遵循由浅入深、由近及远、由感性到理性的原则，结合学生的认知发展特点进行安排。

### **[[《一年级：我眼中的深圳（感知与体验）]]**

**教材内容侧重：** 《古代深圳》中先民的足迹、《深圳民俗文化》中与生活相关的习俗、《近代深圳》中深圳墟的初步印象、《深圳改革开放》中深圳的新变化（图片、地标）。
**年级教学重点：** 激发对深圳的好奇心，初步感知深圳的古老与现代，体验简单的民俗活动。
**年级教学难点：** 理解“古代”、“近代”、“现代”等时间概念的初步含义。
**教学建议：** 多利用直观的文物图片、模型、视频等资源；多组织动手体验活动；鼓励学生用简单的语言表达自己的感受。

**教材内容侧重：** 《古代深圳》中重要的历史文化遗存、《深圳民俗文化》中重要的传统节日和习俗、《近代深圳》中深圳墟的发展、《深圳改革开放》初期深圳的变化。
**年级教学重点：** 初步了解深圳古代和近代的一些重要事件和文化遗存，认识深圳传统文化的多样性，感知深圳的初步发展。
**年级教学难点：** 区分古代和近代的文化遗存，理解墟市在人们生活中的作用。
**教学建议：** 结合博物馆实物进行观察和讲解；引入简单的历史故事和传说；鼓励学生提问和表达自己的想法。

### **三年级：深圳的文化印记（知识积累）**

**教材内容侧重：** 《古代深圳》中商周时期的文化遗存、秦汉时期的城市开端、《深圳民俗文化》中广府和客家民俗、《近代深圳》中反抗外来侵略的斗争、《深圳改革开放》中深圳速度的创造。

**年级教学重点：** 了解深圳古代城市发展的初步历程，认识广府和客家两种主要民系的民俗特点，了解近代深圳人民的反抗精神，初步理解“深圳速度”的含义。

**年级教学难点：** 理解不同历史时期的社会背景，区分广府和客家民俗的异同。

**教学建议：** 讲述历史故事，展示相关的文物和图片资料；引导学生比较不同民俗文化的特点；组织学生进行小组合作探究。

### **四年级：深圳的历史足迹（深入了解）**

**教材内容侧重：** 《古代深圳》中海洋经济的发展、沿海防卫重镇、《深圳民俗文化》中海洋文化习俗、《近代深圳》中香港与深圳的关系、改革开放初期外向型经济的发展。

**年级教学重点：** 了解古代深圳海洋经济和沿海防卫的重要性，认识深圳与香港的历史渊源，理解深圳早期发展外向型经济的策略。

**年级教学难点：** 理解古代海洋经济对深圳的影响，分析香港与深圳近代历史的联系，认识外向型经济的优势和挑战。

**教学建议：** 引导学生阅读历史文献片段；利用地图了解深圳的地理位置；进行情景模拟和小组讨论。

### **五年级：深圳的社会变迁（综合认识）**

**教材内容侧重：** 《古代深圳》中古代移民、《深圳民俗文化》中深圳博物馆馆藏民俗文物精华、《近代深圳》中行政区划和人口变化、海关和铁路的建立、《深圳改革开放》中市场经济体制的建立和发展。

**年级教学重点：** 了解古代深圳移民的来源和影响，认识深圳传统民俗文化的丰富性，理解近代深圳社会结构的变迁，认识改革开放对深圳经济社会带来的深刻变化。

**年级教学难点：** 分析古代移民对深圳文化的影响，理解近代海关和铁路对深圳发展的作用，认识市场经济体制的优势和挑战。

**教学建议：** 引导学生查阅历史文献和地方志；利用图表分析人口和经济数据；组织学生进行社会调查和访谈。

**课程内容（20课时）：**

**第一学期（10课时）：**

| **课时** | **主题**                   | **教学目标**                                                                   | **活动设计**                                                                                     |
| -------- | -------------------------- | ------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------ |
| 1-2      | 远方的客人：古代深圳的移民 | 了解古代广府人和客家人迁徙到深圳的历史和原因；认识不同移民群体带来的文化影响。 | 查阅古代移民迁徙路线图；了解广府和客家人在深圳的分布；讨论移民对深圳文化多样性的贡献。             |
| 3-4      | 民俗的见证：博物馆里的老物件 | 通过观察深圳博物馆收藏的民俗文物，了解不同时期深圳人民的生活习俗和文化艺术。 | 参观深圳博物馆民俗文物展；选择一件感兴趣的展品进行观察和描述；尝试绘制或制作一件类似的“文物”。 |
| 5-6      | 行政区域的变迁：深圳的“版图” | 了解近代深圳行政区划的几次重要变化；认识行政区划对社会管理和发展的影响。       | 查阅近代深圳行政区划变迁的资料和地图；绘制不同时期的深圳“版图”；讨论行政区划变化的原因和影响。   |
| 7-8      | 连接世界：海关和铁路的故事 | 了解近代海关和广九铁路在深圳设立和修建的历史；认识它们对深圳经济发展和对外交流的作用。 | 查找关于深圳海关和广九铁路的历史资料；了解铁路修建的过程和意义；讨论交通设施对城市发展的重要性。 |
| 9-10     | “摸着石头过河”：市场经济的探索 | 初步了解深圳在改革开放中率先进行市场经济体制改革的探索过程和初步成果。   | 了解深圳早期在价格、劳动用工等方面进行的改革尝试；讨论市场经济与计划经济的区别。               |

**第二学期（10课时）：**

| **课时** | **主题**           | **教学目标**                                  | **活动设计**                                                       |
| ------ | ---------------- | ----------------------------------------- | -------------------------------------------------------------- |
| 11-12  | 家族的记忆：广府宗族的发展    | 进一步了解广府宗族在深圳社会发展中的作用；认识宗族文化对广府人的影响。       | 查阅广府宗族的历史资料；了解广府宗族的传统习俗和活动；采访身边的广府人，了解他们的宗族观念。                 |
| 13-14  | 安居乐业：客家围屋的社会功能   | 深入了解客家围屋不仅是居住场所，也是具有社会组织和文化传承功能的社区。       | 分析客家围屋在防御、教育、文化传承等方面的作用；讨论客家人团结互助的精神。                          |
| 15-16  | 人口的增长：深圳的吸引力     | 通过分析近代以来深圳人口数量的变化，了解深圳作为移民城市的特点和吸引力。      | 查阅深圳不同时期的人口统计数据；分析人口增长的原因；讨论人口增长对城市发展带来的机遇和挑战。                 |
| 17-18  | 文化教育的变迁：从私塾到新学堂  | 了解近代深圳文化教育从传统私塾向现代学校转变的过程；认识教育对社会发展的重要性。  | 查找关于近代深圳私塾和新式学堂的资料；比较传统教育和现代教育的特点；讨论教育在社会进步中的作用。               |
| 19-20  | 我的社区调查：深圳社会生活面面观 | 选择自己居住的社区或感兴趣的社会现象进行调查研究，了解深圳现代社会生活的一个方面。 | 学生分组确定调查主题（如社区文化活动、交通出行方式、生活消费习惯等）；进行调查访谈和资料搜集；撰写调查报告并在班级进行分享。 |

**评价方式：**

*   **过程性评价：** 资料查阅分析能力、社会调查和访谈能力、小组合作能力。
*   **终结性评价：** “我的社区调查报告”展示与评价。

### **六年级：深圳的创新发展（思辨与探究）**

**教材内容侧重：** 《近代深圳》中反清革命运动的发展、抗日战争中的深圳、《深圳改革开放》中邓小平两次视察深圳、增创新优势阶段、实践科学发展观阶段。

**年级教学重点：** 了解近代深圳人民的革命斗争和抗日精神，理解改革开放对深圳发展的关键作用和战略意义，认识深圳在不同发展阶段面临的机遇和挑战，培养创新意识和社会责任感。

**年级教学难点：** 分析近代革命和抗日战争对深圳历史的影响，理解不同时期深圳发展战略的调整和意义，培养学生独立思考和批判性思维的能力。

**教学建议：** 引导学生阅读历史文献和名人传记；组织学生进行辩论和案例分析；鼓励学生提出自己的见解和解决方案。

**课程内容（20课时）：**

**第一学期（10课时）：**

| **课时** | **主题**                   | **教学目标**                                                                       | **活动设计**                                                                                               |
| -------- | -------------------------- | ---------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| 1-2      | 革命的星火：近代深圳的反清斗争 | 了解近代深圳人民参与反清革命的一些重要事件和人物；认识革命对中国社会变革的意义。   | 查阅关于三洲田起义等反清革命活动的资料；了解孙中山等革命家的事迹；讨论革命精神的时代价值。                     |
| 3-4      | 烽火岁月：深圳人民的抗日战争 | 了解抗日战争时期深圳沦陷和深圳人民进行抗日斗争的史实；学习抗日英雄的爱国主义精神。 | 观看反映深圳抗战的纪录片片段；了解东江纵队在深圳的抗日活动；讲述抗日英雄的故事；思考和平的来之不易。         |
| 5-6      | 伟大的转折：邓小平与深圳特区 | 了解邓小平倡导建立深圳经济特区的历史背景和战略意义；认识改革开放是中国的伟大抉择。 | 阅读邓小平关于经济特区的讲话片段；了解深圳经济特区建立的过程；讨论建立经济特区对深圳和中国的意义。           |
| 7-8      | “深圳速度”背后的故事：创新与奋斗 | 深入了解“深圳速度”的内涵，认识其背后蕴含的创新精神和艰苦奋斗的品质。         | 分析国贸大厦快速建设的案例；探讨“深圳速度”在不同时期的体现；讨论创新在城市发展中的作用。               |
| 9-10     | 面向世界：深圳外向型经济的发展 | 分析深圳发展外向型经济的优势和挑战；认识对外开放对深圳经济发展的促进作用。     | 搜集深圳外向型经济发展的相关数据；了解深圳知名外贸企业的发展历程；讨论在全球化背景下深圳的发展机遇和挑战。 |

**第二学期（10课时）：**

| **课时** | **主题**                     | **教学目标**                                                                       | **活动设计**                                                                                             |
| -------- | ---------------------------- | ---------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| 11-12    | “二次创业”：深圳发展新阶段 | 了解20世纪90年代以来深圳发展进入“二次创业”阶段的背景和目标；认识深圳发展战略的调整。 | 分析深圳在“二次创业”阶段面临的新挑战和提出的新战略；讨论深圳产业升级和结构调整的必要性。                 |
| 13-14    | 深港合作：携手共进的未来     | 了解深港合作的历史和现状；探讨深港两地在经济、文化等领域的合作前景。               | 查找深港合作的相关案例；分析深港两地合作的优势互补；设计未来深港合作的“金点子”。                           |
| 15-16    | 创新之城：深圳的科技发展之路 | 了解深圳发展高新技术产业的战略和成就；认识科技创新对城市发展的驱动作用。         | 了解深圳高新技术产业发展的典型案例（如华为、腾讯等）；讨论科技创新对人们生活和社会发展的影响。           |
| 17-18    | 可持续发展：建设美好深圳   | 了解深圳在经济发展的同时，在环境保护和城市可持续发展方面所做的努力和面临的挑战。 | 查找深圳在环境保护和可持续发展方面的政策和措施；分析城市发展与环境保护的关系；为建设更美好的深圳提出建议。 |
| 19-20    | 我看深圳未来：我的发展蓝图 | 回顾六年所学，结合自己的观察和思考，展望深圳未来的发展，并提出自己的建议。       | 学生分组讨论对深圳未来发展的设想；选择一个方面（如科技、文化、环境等）设计一份“深圳未来发展蓝图”并在班级进行展示和分享。 |

**评价方式：**

*   **过程性评价：** 参与辩论、案例分析、小组讨论的表现；资料搜集和整理能力；提出问题和解决问题的能力。
*   **终结性评价：** “我看深圳未来”发展蓝图展示与评价。

**实施建议：**

1.  **博物馆资源整合：** 积极与深圳各级博物馆（如深圳博物馆、东江纵队纪念馆、客家民俗博物馆等）联系，开展实地参观、研学活动，利用博物馆的展品和教育资源。
2.  **校内资源利用：** 在学校图书馆、阅览室建立“深圳文化角”，收集与深圳相关的书籍、图片、音像资料，为学生提供学习资源。
3.  **师资培训：** 对参与本课程教学的教师进行培训，提升他们对深圳历史、文化和改革开放成就的了解，掌握博物馆教育的教学方法。
4.  **家校合作：** 鼓励家长参与学生的学习过程，如陪伴孩子参观博物馆、一起阅读相关书籍、分享家庭与深圳的故事等。
5.  **社区联动：** 组织学生参与社区文化活动，如参观社区历史文化遗址、采访社区长者等，拓展学生的学习空间。
6.  **课程评价与反馈：** 定期对课程实施情况进行评价，收集学生和教师的反馈意见，不断完善课程内容和教学方法。
7.  **鼓励学生自主学习：** 引导学生利用网络、书籍等多种资源自主学习，培养他们的探究精神和终身学习能力。

通过以上课程设计和实施建议，期望能帮助深圳的小学生更好地认识自己的家乡，传承深圳的优秀文化和精神，为他们未来的成长奠定坚实的基础。
