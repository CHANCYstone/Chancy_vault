

# 啥是GARCH模型？


想象一下，你正在观察股市，每天的股票价格都在变动。有时候波动很大，有时候又很平静。传统的模型可能觉得这种波动是随机的，但事实并非如此。经验告诉我们，**今天的波动率高，明天可能还会高；今天波动率低，明天可能还会低。** 换句话说，波动率本身并不是常数，它是有“记忆”的，而且会随着时间聚集。

这就是 **GARCH (Generalized Autoregressive Conditional Heteroskedasticity)** 模型要解决的问题。

### 1. 从平均数到波动率

我们知道，一个随机变量通常有两个重要的特征：**平均数（或者叫期望）** 和 **方差（或者叫波动率的平方）**。

- **平均数** 告诉我们变量的中心趋势在哪里。
- **方差** 告诉我们变量离平均数有多远，也就是它的波动程度。

在许多经典的金融模型中，我们假设资产收益率的方差是一个常数。但实际情况是，**方差（波动率）本身也在不断变化**。

### 2. 什么是异方差？

“异方差”听起来很复杂，但其实很简单。

- **同方差 (Homoskedasticity)**：假设所有的波动都一样大。就像你每次扔硬币，出现正反面的概率都是 0.5，每次扔的“不确定性”都是一样的。
- **异方差 (Heteroskedasticity)**：波动大小会变化。就像有时候股市很平静，价格波动很小；有时候股市暴跌或暴涨，价格波动就很大。这种“波动大小会变”的现象，就是异方差。
### 3. ARCH 模型：初窥门径

GARCH 的前身是 **ARCH (Autoregressive Conditional Heteroskedasticity)** 模型。它首次提出，**当前的波动率（或者说方差）是依赖于过去误差的平方的**。

想象一下：你预测明天股票涨跌，如果今天你的预测和实际结果之间出现了一个很大的误差（预测涨，结果跌了很多；或者预测跌，结果涨了很多），那么ARCH模型就会说：“哇，今天的波动好大！这意味着明天的波动可能也会大。”

ARCH 模型就像一个简单的“记忆体”，只记住过去一期或几期的“大波动”，并用它们来预测未来的波动。

### 4. GARCH 模型：更全面的记忆

ARCH 模型虽然进步，但它可能需要很多过去的误差项才能捕捉到波动率的长期依赖性。这就好比你只记住昨天和前天的天气来预测明天的天气，可能不太够。

**GARCH 模型** 更进一步，它不仅考虑过去**误差的平方**（也就是过去“冲击”的大小），还考虑过去**波动率本身**。

它的核心思想是：**今天的波动率（条件方差）不仅取决于昨天的“坏消息”（即误差平方），还取决于昨天的波动率**。

用一个形象的比喻：

- **ARCH** 像是你根据昨天打碎的碗的**数量**来预测今天打碎碗的**可能性**。
- **GARCH** 则是你根据昨天打碎碗的**数量**，**以及昨天你心情有多烦躁**（这导致了打碎碗的**可能性**），来预测今天打碎碗的**可能性**。后者显然考虑得更全面。

**数学表达式（简化版）**：

通常 GARCH(1,1) 模型是金融中最常用的，它的方差方程可以写成这样：

$$\sigma_t^2 = \omega + \alpha \epsilon_{t-1}^2 + \beta \sigma_{t-1}^2$$

这里：

- $$σ_{t}^2​：表示 **今天** 的波动率的平方（也就是方差）。$$
- ω：一个常数项，表示长期平均波动率的一部分。
- ϵt−12​：表示 **昨天** 预测误差的平方，代表了昨天的“冲击”有多大。
- σt−12​：表示 **昨天** 的波动率的平方。
- α：系数，衡量过去**冲击**对当前波动率的影响。如果 α 很大，说明昨天的“坏消息”对今天的波动影响很大。
- β：系数，衡量过去**波动率**对当前波动率的影响。如果 β 很大，说明波动率的“记忆”很强，昨天的波动率高，今天也很可能高。

**1. 核心思想：分解波动率**

GARCH-MIDAS模型的核心在于将传统的GARCH模型的条件方差（波动率）分解为两个部分：

*   **短期波动 (git2)**：反映高频数据（例如，每天的收益率）的短期波动。这部分用标准的GARCH(1,1)模型来描述。
*   **长期趋势 (τt)**：反映低频数据（例如，每月的地缘政治风险指数）的长期趋势。这部分通过MIDAS回归将低频变量（地缘政治风险）与高频波动率联系起来。

**2. 公式解读**

*   **公式 (11): rit = μt + √(σit2)εt**
    *   `rit`：第t个月第i天的资产收益率。
    *   `μt`：条件期望，可以理解为收益率的预期值。
    *   `σit2`：条件方差，也就是波动率，是GARCH-MIDAS模型要重点分析的对象。
    *   `εt`：误差项，服从标准正态分布。
    *   这个公式表明，每天的收益率由预期收益和波动率决定。

*   **公式 (12): σit2 = git2τt**
    *   `σit2`：总的波动率。
    *   `git2`：短期波动部分，由GARCH(1,1)过程决定。
    *   `τt`：长期趋势部分，由低频变量（地缘政治风险）决定。
    *   这个公式是GARCH-MIDAS模型的关键：将波动率分解为短期和长期两部分。

*   **公式 (13): git2 = m + αεi-1,t2 + βgi-1,t2**
    *   `git2`：短期波动率。
    *   `m`：常数项。
    *   `α`：衡量前一天的误差项（冲击）对当前波动率的影响。
    *   `β`：衡量前一天的波动率对当前波动率的影响。
    *   这个公式是标准的GARCH(1,1)模型，描述了短期波动率的自相关性。

*   **公式 (14): τt = m + θ∑k=1KΦk(ω1, ω2)RVt-k**
    *   `τt`：长期趋势。
    *   `m`：常数项。
    *   `θ`：长期趋势对波动率的影响系数。
    *   `RVt-k`：已实现波动率（Realized Volatility），一种衡量历史波动率的指标。
    *   `Φk(ω1, ω2)`：Beta权重函数，用于给不同滞后期的已实现波动率赋予不同的权重。
    *   `K`：最大滞后期数。
    *   这个公式描述了长期趋势如何受到过去K期的已实现波动率的影响。权重函数`Φk(ω1, ω2)`决定了不同滞后期波动率的影响程度。
