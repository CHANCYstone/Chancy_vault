# 📒 Note

首先，开宗明义地讲，物质基础决定上层建筑（这里我没有采用经济基础，是因为我认为，纯粹的经济基础是不足够的，应该包含武器装备的计算）。这就是这个世界运行的规则。这是”落后就要挨打的道理“。1840年，八国联军侵华，是因为八国联军的物质基础够强。甲午战争，中国割让台湾岛给日本殖民，也是因为日本军事的强大。1931年，918事变日本侵华，也是因为日本的强大。而近些年来，美国能不断地发动阿富汗战争而没人能制裁的了他，也是因为他足够强大。能把日本打进失去的20年，也是因为美国的强大。因此，我认为，台湾问题，必将随着中国的强大而得到解决。

台湾人想要什么？
现在来看，台湾人想要独立。这恐怕是大多数台湾人的意志。但为什么对于中国来说，台湾这块地就属于中国呢？合理性在于什么地方？很多人喜欢拿着中国从来没有占领台湾说事，说台湾只被中华民国占领过。好，我们就基于这个基础讨论。假设现在台湾穷兵黩武，拥有原子弹，而中国因为ccp的管理不善积贫积弱，成为东亚病夫。我想就没有所谓的台湾这个称呼，只有一个称呼叫做”中华民国“，并且这个”中华民国“将要收复大陆，去解救亿万万水深火热的中国人民，去成为”中国“的代名词，这将是任何一个台湾人民的心意。因此，事实上没有”台湾问题“，只有谁代表着中国的问题。然而，事实是，ccp把中国治理的很好，经济强，武德也充沛，而中华民国，作为资产阶级政府，虽有”三民主义一统中国“的野心，却没有“三民主义一统中国”的决心。他的软弱性，决定了他物质基础的弱势地位。随着中国国力的不断壮大，看到“一统中国”无望之后，才有所谓的”台独问题“的出现——希望在竞争中求得逃避。而这是中国绝对不可能允许出现的事情。
