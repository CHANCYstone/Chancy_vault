**三、技术突击重点（20分钟）**

  

**📌 SQL 高频题型**

• 窗口函数（如：ROW_NUMBER() OVER (PARTITION BY...)）

• 多表JOIN（内连接、左连接区别，过滤顺序）

• 数据去重（DISTINCT vs GROUP BY）

• 分组聚合（比如求每个国家月度新增用户数）

  

> 示例题：找出每个国家上个月新增用户数最高的那天

```
SELECT country, date, new_user_count
FROM (
  SELECT country, date,
         COUNT(DISTINCT user_id) AS new_user_count,
         ROW_NUMBER() OVER (PARTITION BY country ORDER BY COUNT(DISTINCT user_id) DESC) as rn
  FROM user_log
  WHERE date BETWEEN '2025-03-01' AND '2025-03-31'
  GROUP BY country, date
) t
WHERE rn = 1;
```

**📌 大数据开发常识**

• 数据仓库分层（ODS→DWD→DWS→ADS）

**四、支付业务理解（10分钟）**

  

**💡 支付业务理解点**

• 国际支付：涉及**时效性**、**安全性**、**合规性**、**多币种/多地域**。

• 数据挑战：高并发、大规模（PB级别）、一致性保障

• 技术挑战：事件驱动设计、幂等处理、实时反欺诈

• AI探索点：利用 LLM 做风险识别、自动建模、图分析欺诈
