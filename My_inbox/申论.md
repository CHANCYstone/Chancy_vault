[申论](https://zhida.zhihu.com/search?content_id=655308604&content_type=Answer&match_order=1&q=%E7%94%B3%E8%AE%BA&zhida_source=entity)这一科，其实非常强调概括归纳能力，和文字表达能力。答题格式，其实是最简单、最容易掌握的部分。下面干货，大家收藏好，基本能解决你所有对格式的困惑：

## **一，申论都考哪些题型？**

要将格式，首先你要具备区分题型的能力。否则你连在考什么都不知道，怎么用对应的格式？  
  
首先，申论考的是答题要素。什么叫**答题要素**呢？**问题、对策、影响、原因、态度、事例**，这些关键词信息，就是出题人想让你写的要素。当然，有时候他不明写，但你要能读出来。  
  
这些要素，出题人到底想考什么？我总结了一个表，大家可以自行保存：

|   |   |   |   |   |   |
|---|---|---|---|---|---|
|问题|影响|原因|对策|态度|事例|
|负面信息|好的结果（意义）坏的影响（危害）|好的原因（积极）坏的原因（消极）|动词+名词|观点、看法|论据充实|

这些答题要素，排列组合，就是你要写出来的内容。  
而题目里，有的题问得很直接，有的题问得不直接。我们根据题目的问法，又把申论题型分四类：**[单一题](https://zhida.zhihu.com/search?content_id=655308604&content_type=Answer&match_order=1&q=%E5%8D%95%E4%B8%80%E9%A2%98&zhida_source=entity)、[综合题](https://zhida.zhihu.com/search?content_id=655308604&content_type=Answer&match_order=1&q=%E7%BB%BC%E5%90%88%E9%A2%98&zhida_source=entity)、公文题、[写作题](https://zhida.zhihu.com/search?content_id=655308604&content_type=Answer&match_order=1&q=%E5%86%99%E4%BD%9C%E9%A2%98&zhida_source=entity)**。下面咱们分别来讲。  
  
什么是**单一题？**  
  
就是题干直接告诉你，我要考什么。比如去年陕西省考这道题：

![](https://pica.zhimg.com/80/v2-e74445affc58121cc9fe2f9c48f723c2_1440w.webp?source=1def8aca)

直截了当，让你分析“原因”，你都不用费心去揣摩他要考啥。这种**提问明确的题目**，**就是单一题**。  
  
什么是**综合题**？  
  
对，提问要素模糊，或者一个问题包含多个要素，就是综合题。很多题目中让你**谈谈理解**，**谈谈看法**，就是综合题。  
  
比如河南的这道题：

![](https://pica.zhimg.com/80/v2-42ced0607b03de738b8e9c13289920af_1440w.webp?source=1def8aca)

**公文题**，顾名思义就是让你写公文啦，一般题目会明确说，让你写一份、一张、一个+发言稿、宣讲稿、倡议书、通知、新闻稿等等，“**数量词+文种**”，就是公文题，非常好判断。  

![](https://pica.zhimg.com/80/v2-3fc7914e34e8e11425b5f498a5f31d99_1440w.webp?source=1def8aca)

公文题一般都这么出，至今没看到有例外

最后最重要的**写作题**，也就是大家常说的大作文，一般要求就是“参考资料，联系实际，自选角度，自拟题目，写一篇议论文”。按照议论文的格式写就好。  
  
读懂了题目要考啥，如何作答，我这里就不展开了。因为申论的答题，一篇两篇文章也讲不清楚。这需要系统地去学习，去练习，去动笔上手做真题。大家想深入学习、高效率提分的话，我给大家推荐一个课：[公考优师领学计划](https://zhida.zhihu.com/search?content_id=655308604&content_type=Answer&match_order=1&q=%E5%85%AC%E8%80%83%E4%BC%98%E5%B8%88%E9%A2%86%E5%AD%A6%E8%AE%A1%E5%88%92&zhida_source=entity)，去听一听，你会获益匪浅。

## **二，申论不同题型答题格式示范**

讲了题型，下面来说最主要的部分——答题格式。我特地用打印机扫描了一下**我用的答题卡**，给大家展示：

![](data:image/svg+xml;utf8,<svg%20xmlns='http://www.w3.org/2000/svg'%20width='578'%20height='442'></svg>)

看到没有，小题和大题，都有字数提示。真正的考场上，字数提示其实没有这么密集，但要求你写300字，他就一定会在300字的时候提醒你。上图右侧那张是大作文的格子，注意大作文写超了是没问题的。小题一般格子只会多给100。  
  
下面我给大家展示一下不同题型的答题格式。  
  
首先说**单一题**，你已经通过提示词（困难、不足、瓶颈、问题、挑战、难点、缺失等等）了解了题型，知道它想问什么了。同时读题后也归纳好答案要点了。那么怎么开始动手写呢？  
  
**单一题**组织答案，不用空行，直接顶格开始写。你可以用大点带小点的形式，比如：一，巴拉巴拉，1，xxxx；2，xxxx；3，xxxx；二，……  
  
也可以直接1，xxxx，2，xxxx，3，xxxx。  
  
申论是按词赋分，怎么确定自己应该写多少词语呢？有个公式：**单一题总字数/5×2/3（总字数除以5乘以三分之二）**。150字的题，可以写20个词，200字写25个词。如果要求不超过350字，给了500个格子，也必须按照350字写。  
  
写的时候，字数要均衡，比如20个词，分5条，那么每条4个词左右即可。不要过多也不要过少。  
  
**综合题**，格式和单一题是一样的。因为都是概括归纳。有的综合题其实包含两问，比如先问问题，再问对策，那么就分开写：问题：1234,；对策：1234。

![](data:image/svg+xml;utf8,<svg%20xmlns='http://www.w3.org/2000/svg'%20width='622'%20height='594'></svg>)

**公文题**，格式就要严谨一些了。  
  
首先说标题，有的题目会给定标题，那就按题目来，大多题目不给标题，要先确定文体。公文标题通常是“**关于+主题+的+文种**”，这样写最保险。可以参考《党政机关公文处理工作条例》。  
  
其次，书写格式有几个重点。首先是**称谓**，上行文下行文的称谓是不一样的，要特别注意，别闹出对着领导叫小弟的笑话。称谓就是“**主送机关**”，比如“xx县人民政府”、“各位社区居民”、“各位网民朋友”等等。**称谓顶格书写，并加冒号**。  
  
**落款**，发文机关/署名一般要写全称，**成文日期**，要用**阿拉伯数字+年月日**。在正文右下方，先给成文日期右边留出4个空格子出来，再把发文机关写到日期上一行，和日期居中对齐。

写作题，格式和议论文的相同的，大家从小就学了。注意，分段不要过多，4—6段即可，事例不要过长，你不是写评书剧本呢。要论述和事例平分秋色，最好更重论述。  
  
写作题千万注意，不要跑题，题目跑、正文跑，都是很大的扣分项。这里特别给大家讲一下，**写作题的评分标准**。很多同学都觉得申论考试是玄学，明明觉得答的不错，结果分数却不高。其实就是不太了解评分标准。

## **三，答题时要注意的小细节**

1.**错别字问题**。发现错别字，不要涂抹出个墨疙瘩，一条斜线划掉即可。如果发现不了错别字，一般来说，**集齐3个错字扣1分**，偶然一个没关系，但是会很影响阅卷老师的观感。  
  
2.**字数问题**。字数超字数一般不扣分，但要注意题目要求。刚在3月16号考完的24年陕西省考中，倒数第二题让写“发言提纲”，要求字数在500字**以内**，结果格子给了600多个，很多同学写超了，据说也会扣分。据说超10个字扣1分，最高扣3分。  
  
但是，宁愿多写一点，不要少写。比如大作文，如果让写800字，不足800字的，每少50字扣1分，最多扣2分，**不足500字，直接打成三类文**。  
  
3.**格式问题**。一般答非公文题，开头不用空两格，也不用像中学生一样写个“答”，你已经过了举手上厕所的年纪了。  
  
4.**标点问题**。标点最好占一格，包括” “、《 》。分条写的时候一、二、三、①、②、③，都要占一格。公文写作最好写第一、第二、第三。小题不用注意格式，都是踩点给分的，写完一点，不必另起一行，每那么多格子。

单独占一格的标点符号，常见的有：逗号、分号、冒号、问号、叹号、句号、顿号等；需要占双格的标点符号，常见的有：破折号、省略号、引号、书名号、括号等。  
  
除了《 “ --这三种标点符号，其他的标点符号包括 》”一律不能顶格写，写不下也要挨着格子边写。

![](data:image/svg+xml;utf8,<svg%20xmlns='http://www.w3.org/2000/svg'%20width='615'%20height='636'></svg>)

5.最重要的一点——一定要字迹清晰，**字迹清晰、让阅卷老师看着舒服、看得懂，比字迹漂亮更加重要**！现在传出来的阅卷要求是：尽量减少字迹对成绩的影响。注意是“尽量”。所以你写不漂亮，问题不大，但是让老师看半天也看不清，影响了阅卷老师的心情，那就危险了。

**1、标点符号需要独立占格**

这是标点符号最基本的格式，很多同学习惯于将标点符号点在字的右下角，平时不是很显眼。但是，公务员考试是有格子的，而标点符号的格式在评分标准中有明确的要求，所以必须习惯于将标点符号也当成文字的一个独立部分。

单独占格的标点符号常见的有：逗号、分号、冒号、问号、叹号、句号、顿号等。需要占双格的标点符号常见的有：[破折号](https://zhida.zhihu.com/search?content_id=459594595&content_type=Answer&match_order=1&q=%E7%A0%B4%E6%8A%98%E5%8F%B7&zhida_source=entity)、[省略号](https://zhida.zhihu.com/search?content_id=459594595&content_type=Answer&match_order=1&q=%E7%9C%81%E7%95%A5%E5%8F%B7&zhida_source=entity)、[引号](https://zhida.zhihu.com/search?content_id=459594595&content_type=Answer&match_order=1&q=%E5%BC%95%E5%8F%B7&zhida_source=entity)、[书名号](https://zhida.zhihu.com/search?content_id=459594595&content_type=Answer&match_order=1&q=%E4%B9%A6%E5%90%8D%E5%8F%B7&zhida_source=entity)、[括号](https://zhida.zhihu.com/search?content_id=459594595&content_type=Answer&match_order=1&q=%E6%8B%AC%E5%8F%B7&zhida_source=entity)等。

对于双格标点符号，一定要注意即使是多个格子也依然要将其写成一个整体。但如果遇到以下两种特殊情况，则需要做特殊处理。

**2、标点符号的特殊格式**

1、除了《 “--这三种情况，其他的标点符号一律不能顶格写，如果在末格写完文字需要除以上三种情况之外的标点符号，就在字的右下角标出；破折号和省略号需要占双格，但如果出现在最后一个，就只需要占一格即可，不能将其分成两个部分。

2、《 " 这两种情况不能在末格出现，如果出现在末格，或者与文字挤在一格，或者移到下一行出现。

**二、** **数字的格式**

对于一个两位及以上的独立数字来说，两位数字占一个格子，例如2022，就是"20"一格，“22”一格；奇数位数的数字，同样适用，最后空一个字符即可，如20000，最后一个出现一个“0”，后面空着，以此类推。

但不管是哪种情况，如果此数字是作为一个整体出现的，那就必须保持其形式上的完整性，靠近写出。

**三、** **写不写“答”**

几乎所有考生都会问这个问题，其实在改卷的时候，没有哪位同学是因为写了“答”字而多得了几分，也没有哪位同学因为没有写而被扣掉几分。

所以结论就是，写不写都可以，只要保持整个答案的格式一致性即可。

如果写，那就是"答"和其后面的冒号占两格，顶格写，后面分段直接空两格开行。如果不写，开行的格式和后面分段开行对齐。

**四、** **标题的格式**

标题只要大致写在中间就可以了。如果有副标题，在正标题下面空两格(正标题文字的两格)写破折号，起写副标题，但是要将正副标题作为一个整体写在中间。

一般不提倡大家使用太长的标题，但如果标题过长，分两行书写，一般格式是上一行长一些，下一行短一些，两两居中对齐。对称式的标题或者中间空一格写成一行或分两行书写。
