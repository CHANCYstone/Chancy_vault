- [[年志]] 黄色

- ![[Pasted image 20240327195040.png]]

- 首先，你应该会发现我的目标并不是用 Check box ，原因是我想要**能一眼就区分出「任务」和「目标」的差别**。在[之前的文章中](https://drawwow.com/new-year-resolution/)提到我会 **==将目标变成可执行的任务，而这些任务就会使用 Check box，只要完成就打勾==**。

- 而目标则是用 emoji 的✅来表示，这样一眼就能清楚分辨任务与目标。

- 右边是我的目标心智图，**透过心智图可以很好的梳理思绪**，帮助自己知道如何[将目标拆解成任务](https://drawwow.com/new-year-resolution/#task)，甚至是不同目标间的关联性。

- 在下方我放上了上次[回美国](https://drawwow.com/2022usa-0/)时学到的方法，将一年拆解成4个 Quarter，我用 Q 表示。一个 Q 就是三个月，4 个 Q 刚好一年。

- 我先简单地在 **==每个 Q 中写下要专注的目标，还有即将要发生的事情==**，有点像未来志。

- 我会在每个 Q 结束时**复盘、回顾自己是否有完成必要的任务、离目标还有多远**。

- 月志 红色

- ![[Pasted image 20240327195234.png]]

- 月志只有两个部分：**目标**、**回顾**。

- ✅目标可以直接参考年志，并且选几个想在这个月完成的目标。

- 回顾则是我以前还在手写时就有的习惯，在每个月底时，我会重新检视我当月的子弹笔记，并且**把每一天用一句话说明**。

- 这个动作对我来说非常重要，因为 ==**人们往往高估自己一天能做多少事，却低估长期下来能做多少事情。**==

- 日志是白色

- 在日志中，我固定纪录的元素跟我以前在手写子弹笔记时一样。

- [ ] 任务

- 👍 肯定

- ❤️ 感恩

- 🔧 反省

- ==**任务比起用纸本手写来说方便太多了**！==

- 以前用纸本时如果遇到没有完成的任务，还要重写一次，而在 Heptabase 中，只要点一下它的日期，就可以直接将任务移到指定的日子，非常方便！

- #❓ 考虑引用 task 任务插件

- 还可以在左边的侧边栏点开 Task ，就可以看到所有的任务以及指派的日期，就像会自动同步的未来志一样，非常方便。

- ![[Pasted image 20240327201334.png|388]]

- #❓ 讲日志放到一个面板上去（考虑在Obsidian 中怎么实现）

- 我们可以马上看到哪一天有什么事情，可以**用鸟瞰的角度看我们的过去、现在、未来**。

- 总结

- 电子版的子弹笔记「**更少阻力**」

- Obsidian 是实现[[🔗 子弹笔记]]的最小成本行动

- 我完全不用想「这则笔记要归类到哪？」，而是只要**有灵感、有想法，全部丢到日志中**。

- 我也不会因为想把手写子弹笔记做得美美的而感到备感压力，在用 Heptabase 做子弹笔记后，我感觉得到了解放。
