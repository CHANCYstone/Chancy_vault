# 5-如何设置目标？如何安排好自己的事情

有时候我知道该[[如何设置目标]] （还需要再读读书）

但更多时候，我不知道该如何设置目标。

心中有那么一个list 但感觉这个list是无限大的。

# 10-压力时刻

在压力大闷的时候问自己几个简单的问题：

1. 自己的世界会因为这件事情毁灭掉么？
2. 这件事情能不能有解决方案？
3. 怎么解决，关键的点是什么？
4. 回顾以前，自己是不是付出所有来解决这件事情了？
5. 自己是不是再也不想要这样的事情再发生一次了？
6. 怎么解决问题？
